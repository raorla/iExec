# dapp development

Some examples to use iexec for dapp development

## install

```sh
npm ci
```

## usage

create a `.env` file following [`.env.template`](.env.template)

run the scripts

```sh
npx tsx scripts/<script-name>
```

hackable examples:

- `deploy-app.ts`
- `publish-apporders.ts`
- `run-app.ts`
- `run-app-result-encryption.ts`
